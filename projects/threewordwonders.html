This project was a fun way to experiment with Semantic UI, refresh my memory of knockout, and play with some new api's such as flickr, Google static maps, and what3words. I converted a csv dump of the UNESCO World Heritage Sites into json and using two-way binding via knockout randomly cycle through the list while fetching flickr photos, maps, and more.
