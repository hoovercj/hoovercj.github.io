VT-Graph Reconfigurable Real-time Graphing of Physiological Data

Overview

VT-Graph is an android application I developed for the VirtuTrace project under the direction of Drs. Nir Keren and Warren Franke. The system is used for collecting physiological data from participants during simulations with soldiers, law enforcement officers, and firefighters in virtual reality environments. The participant is hooked to Biopac which monitors physiological data such as ECG, cardiac output, respiration rate, and galvanic skin response which is sent to an Android device running VT-Graph. The application allows observers to create reconfigurable dashboards to display physiological data numerically, in line graphs, or in bar graphs.

Technologies Used

Android, VRPN, 3rd Party Libraries (GraphView, AChartEngine, SlidingMenu)

Skills Developed

Software Design Patterns, Client-Server Architecture, Exposed to JNI
